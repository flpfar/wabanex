# Wabanex

> An API to manage your Gym trainings.

## Technologies
  * Elixir
  * Phoenix
  * GraphQL

## Running

  * Install dependencies with `mix deps.get`
  * Create and migrate your database with `mix ecto.setup`
  * Start Phoenix endpoint with `mix phx.server`
  * Access `http://localhost:4000`

## Testing
  * Run `mix test`
